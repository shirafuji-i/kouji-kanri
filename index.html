☁ クラウド同期設定（PC・携帯共有）

【手順①】Realtime Database の作成

  1. console.firebase.google.com を開く
  2. 「プロジェクトを追加」→ 任意の名前で作成（Analyticsは不要）
  3. 左メニュー「構築」→「Realtime Database」をクリック
  4. 「データベースを作成」→ 場所はデフォルトでOK → 「テストモード」を選択して有効化
  5. 作成後に表示される https://xxx-default-rtdb.xxx.firebasedatabase.app の
    URLをコピーして下の「データベースURL」欄に貼り付け

【手順②】アプリの設定情報を取得

  1. 左メニューの歯車アイコン →「プロジェクトの設定」
  2. 下にスクロール →「マイアプリ」→ </>（ウェブ）をクリック
  3. アプリ名を入力（何でもOK）→「アプリを登録」
  4. 表示された apiKey authDomain など全体をコピーして
    下の「設定JSON」欄に貼り付け
